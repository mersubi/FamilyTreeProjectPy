# 🌳 Family Tree Project

Интерактивное генеалогическое древо с современным темным неоновым дизайном, написанное на Python с использованием библиотеки [NiceGUI](https://nicegui.io/).

## 🌟 Возможности
- **Умное управление родственниками**: Интеллектуальная система добавления связей (родители, дети, братья/сестры).
- **Визуализация**: Автоматическая раскраска поколений и линий связей на SVG-схеме.
- **Фотогалерея**: Просмотр и управление галереей фотографий для каждого члена семьи.
- **Сохранность**: Автоматическое сохранение данных в локальный файл `database.json`.
- **Контейнеризация**: Проект полностью упакован в Docker для быстрого и безболезненного запуска.

---

## 🚀 Как запустить проект?

### Способ 1: С помощью Docker (Рекомендуемый)
Для запуска потребуется установленный [Docker](https://www.docker.com/) и Docker Compose.

1. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   git clone https://github.com/mersubi/FamilyTreeProjectPy.git
   cd FamilyTreeProjectPy
   ```

2. Запустите контейнер в фоновом режиме:
   ```bash
   docker-compose up -d --build
   ```

3. Откройте браузер и перейдите по адресу: [http://localhost:8080](http://localhost:8080)

*💡 Примечание: База данных (`database.json`) и загруженные фотографии (`media/`) сохраняются локально на вашем компьютере, поэтому данные не пропадут при перезапуске или обновлении контейнера.*

---

### Способ 2: Классический запуск (Python)
Убедитесь, что у вас установлен Python версии 3.9 или выше.

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/mersubi/FamilyTreeProjectPy.git
   cd FamilyTreeProjectPy
   ```

2. *(Опционально)* Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для macOS/Linux
   # venv\Scripts\activate   # Для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите проект:
   ```bash
   python3 main.py
   ```

5. Проект автоматически откроется в вашем браузере!
